# /workflows-compound

记录解决的问题和学到的经验，让知识复合增长。

## 理念

**复合工程**：每一次工程工作都应该让后续工作更轻松。

## 流程

1. **问题回顾** - 描述遇到的问题
2. **解决方案** - 记录如何解决
3. **关键学习** - 提炼可复用的经验
4. **更新文档** - 将学习写入项目文档

## 记录模板

```markdown
## [日期] [问题标题]

### 问题
[描述遇到的问题]

### 原因
[根本原因分析]

### 解决方案
[如何解决的]

### 学习
[可复用的经验]

### 预防
[如何避免再次发生]
```

## 输出位置

- 项目级学习 → `docs/learnings/` 或 `AGENTS.md`
- 通用模式 → 考虑创建新的 Skill

## 后续步骤
